# Find Your Campus 🎓

Building the bridge between high school graduates and their future.

### The Problem
Most students choose colleges based on fancy websites and placement brochures that often hide the ground reality. By the time they realize the truth, it's too late.

### Our Solution
A direct line to seniors. We connect aspirants with current students at their target campuses so they can ask the questions that actually matter:
- "What's the *real* placement scenario for average students?"
- "How's the coding culture? Any active communities?"
- "Is the hostel food actually edible?"
- "Which professors should I avoid?"

---

## Tech Stack (The "Move Fast" Stack)
- **Frontend**: React + Vite (Fast builds, clean code)
- **Styling**: Tailwind CSS (Rapid UI development)
- **Animations**: Framer Motion (Premium feel)
- **Backend**: Node.js + Express (Scalable and quick to setup)
- **Database**: Supabase (Postgres with less headache)

---

## Project Structure
We keep it clean and modular:
- `frontend/`: The React application.
- `backend/`: Express API for handling leads and payments.
- `docs/`: Our strategy and roadmap docs.

## Setup Instructions

### Backend
1. `cd backend`
2. `npm install`
3. `node server.js` (Runs on port 5001)

### Frontend
1. `cd frontend`
2. `npm install`
3. `npm run dev`
